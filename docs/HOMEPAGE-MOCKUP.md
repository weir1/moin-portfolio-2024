## Animation Notes

1. **Scroll Animations**
```markdown
- Hero text: Fade in + slide up
- Metrics: Counter animation
- Cards: Reveal on scroll
- Images: Subtle parallax
```

2. **Interactive Elements**
```markdown
- Nav: Glass effect on scroll
- Buttons: Hover scale + glow
- Cards: Hover lift + shadow
- Form: Smooth focus states
```

3. **Page Load**
```markdown
- Progressive content reveal
- Smooth metric counting
- Gentle fade-ins
- Background gradient shift
```
